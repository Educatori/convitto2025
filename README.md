🏨 Registro Dinner e Convitto 25-26

Questo progetto è un'applicazione web per la gestione dei turni mensa e dei permessi degli studenti del **Convitto**. Permette di monitorare le presenze al 1° e 2° turno di cena e gestire i permessi permanenti.

## 🚀 Funzionalità principali
* **Gestione Turni:** Selezione rapida tra 1° Turno (18:30) e 2° Turno (19:15).
* **Ricerca Studenti:** Filtro dinamico per nome o numero di stanza.
* **Permessi Permanenti:** Pannello laterale per la consultazione dei permessi d'uscita.
* **Stampe:** Generazione di report pronti per la stampa per il Convitto e il Dinner.
* **Daily Reset:** Funzione per pulire i dati giornalieri e ricominciare la rilevazione.

## 📂 Struttura del Progetto
* `index.html`: Pagina principale con i link di navigazione.
* `dashboard.html`: Interfaccia operativa del registro.
* `/css`: Contiene i fogli di stile (layout, colori e responsive design).
* `/js`: Contiene la logica JavaScript e il database degli studenti.

## 🛠️ Note Tecniche
Il sistema utilizza il **LocalStorage** del browser per salvare i dati. Questo significa che le informazioni inserite rimangono salvate sul dispositivo utilizzato senza bisogno di un server esterno.
